# Twinkle

Разработка высоконагруженной социальной сети с использованием микросервисной архитектуры.

# Архитектурные требования
+ Микросервисная архитектура с четким разделением ответственности
+ Асинхронная коммуникация между сервисами
+ Горизонтальное масштабирование каждого сервиса
+ Кэширование для снижения нагрузки на БД
+ Поисковый движок для контента
+ Очереди задач для фоновых операций
+ REST API + WebSocket для реального времени

# Стек технологий
+ API Gateway: FastAPI
+ Сервисы: FastAPI, SQLAlchemy, Pydantic
+ Базы данных: PostgreSQL, Elasticsearch, Redis
+ Очереди: RabbitMQ + Celery
+ Кэш: Redis
+ Контейнеризация: Docker + Docker Compose / Kubernetes
+ Мониторинг: Prometheus + Grafana
